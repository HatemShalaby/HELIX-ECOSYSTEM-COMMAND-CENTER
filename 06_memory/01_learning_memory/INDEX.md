# Learning Memory Index

## Purpose

Tracks learning progress, confusion, lesson quality, quiz performance, weak spots, and next learning recommendations.

This memory category supports the Education Center and helps the system adapt future lessons to the user's actual learning state.

## What Belongs Here

- lessons completed
- concepts understood
- concepts that caused confusion
- quiz results
- weak spots
- useful explanations
- failed explanations
- next recommended learning steps

## What Does Not Belong Here

- engine forecast results
- client business patterns
- AI/tool failures
- architecture decisions
- raw transcripts

## Record Format

Each record should include:

- Date
- Topic
- Lesson source
- What was learned
- What was confusing
- Quiz result
- Weak spot
- Next recommended lesson
- Confidence level

## Suggested File Naming

```text
YYYY-MM-DD-topic-name.md
```

Example:

```text
2026-06-22-demand-units.md
```
